# 이상한 문자 만들기

```python
def solution(s):
    answer = ''
    cnt = 0
    for i in range(len(s)):
        if s[i] == ' ':
            cnt = 0
        else:
            cnt += 1

        if cnt % 2 != 0:
            answer += s[i].upper()
        elif cnt % 2 == 0:
            answer += s[i].lower()
    return answer
```
