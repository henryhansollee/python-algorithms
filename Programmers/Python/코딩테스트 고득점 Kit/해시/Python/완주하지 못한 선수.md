# 완주하지 못한 선수

###### 나의 풀이

```python
def solution(participant, completion):
    answer = participant[-1]
    n = len(completion)
    
    participant.sort()
    completion.sort()
    
    for i in range(n):
        if participant[i] != completion[i]:
            answer = participant[i]
            break
    
    return answer
```

###### 다른 풀이

```python
import collections

def solution(participant, completion):
    answer = collections.Counter(participant) - collections.Counter(completion)
    return list(answer.keys())[0]
```
