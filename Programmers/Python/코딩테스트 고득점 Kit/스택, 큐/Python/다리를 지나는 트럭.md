# 전화번호 목록

###### 나의 풀이

```python
def solution(phone_book):
    n = len(phone_book)
    
    phone_book.sort()
    for i in range(1, n-1):
        if phone_book[i-1] in phone_book[i] or phone_book[i] in phone_book[i+1]:
            return False
    
    return True
```

###### 다른 풀이

```python
def solution(phone_book):
    phone_book.sort()
    
    for p1, p2 in zip(phone_book, phone_book[1:]):
        if p2.startswith(p1):
            return False
            
    return True
```
