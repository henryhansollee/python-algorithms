# 최대공약수와 최소공배수

```python
def solution(n, m):
    nums = list(range(1, n*m+1))
    a, b = -999999999, 999999999
    for num in nums:
        if n % num == 0 and m % num == 0:
            a = max(a, num)
        if num % n == 0 and num % m == 0:
            b = min(b, num)
    answer = [a, b]
    return answer
```
