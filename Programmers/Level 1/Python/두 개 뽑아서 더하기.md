# 두 개 뽑아서 더하기

```python
def solution(numbers):
    answer = []
    n = len(numbers)
    
    for i in range(n):
        for j in range(n):
            if i != j:
                answer.append(numbers[i] + numbers[j])
    
    answer = list(set(answer))
    answer.sort()
    
    return answer
```
