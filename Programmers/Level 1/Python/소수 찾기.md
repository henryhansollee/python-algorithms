# 소수 찾기

```python
def solution(n):
    answer = 0
    check = [False] * (n+1)
    
    for i in range(2, int(n ** 0.5) + 1):
        if check[i] == False:
            for j in range(i+i, n+1, i):
                check[j] = True

    for i in range(2, n+1):
        if not check[i]:
            answer += 1
            
    return answer
```
