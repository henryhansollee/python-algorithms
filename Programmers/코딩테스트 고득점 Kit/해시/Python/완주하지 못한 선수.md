# 완주하지 못한 선수

```python
def solution(participant, completion):
    
    participant.sort()
    completion.sort()
    
    c_len = len(completion)
    
    answer = participant[-1]
    
    for i in range(c_len):
        if participant[i] != completion[i]:
            answer = participant[i]
            break
    
    return answer
```
