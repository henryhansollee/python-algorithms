# 전화번호 목록

```python
def solution(phone_book):
    phone_book = sorted(phone_book)

    for p1, p2 in zip(phone_book, phone_book[1:]):
        if p2.startswith(p1):
            return False
        
    return True
```

## 1. zip()

- 같은 인덱스끼리 잘라서 리스트로 반환



## 2. startswith()

- 찾고 싶은 문자열의 형태가 처음에 있을 때



## 3. endswith()

- 찾고 싶은 문자열의 형태가 마지막에 있을 때
