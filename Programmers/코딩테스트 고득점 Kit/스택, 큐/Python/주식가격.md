# 위장

###### 나의 풀이

```python
def solution(clothes):
    
    n = len(clothes)
    
    c_dict = {}
    for i in range(n):
        if clothes[i][1] in c_dict:
            c_dict[clothes[i][1]] += 1
        else:
            c_dict[clothes[i][1]] = 2
            
    answer = 1
    c_dict_vals = list(c_dict.values())
    for i in range(len(c_dict_vals)):
        answer *= c_dict_vals[i]
    
    return answer - 1
```

###### 다른 풀이

```python
def solution(clothes):
    from collections import Counter
    from functools import reduce
    cnt = Counter([kind for name, kind in clothes])
    answer = reduce(lambda x, y: x*(y+1), cnt.values(), 1) - 1
    return answer
```
